# BambaraTokenizer

A Python library for tokenizing and processing Bambara text. This library includes various methods for tokenization, stopword removal, special token handling, and more.

## Features

- Tokenization with special character handling
- Stopword removal
- Vocabulary management
- N-grams and padding
- BPE tokenization
- Synonym replacement
